# 自動溶着ステージ

超音波溶着機を動かすための自動溶着ステージを制作したので、共有します。

## ファイルの説明
* **公開版.step**
  CADデータです。
* **自動ステージ用部品.xlsx**
  購入部品と3Dプリンターで製作した部品の一覧です。
* **使い方.pptx**
  私の環境で動かすときのマニュアル資料です。参考までに。
* **回路図.pptx**
  どのような配線をしたのかを記載しています。
* **その他 (.py, .json)**
  プログラムファイル群です。設定ファイル（.json）含め、すべて同じディレクトリで使用してください。
  ※ `foot_button.py`は溶着機をフットペダルでオンオフできるようにするものです。単独で実行するものではなく、`main_app.py`に組み込まれて動作します。
