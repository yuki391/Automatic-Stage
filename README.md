# 自動溶着ステージ

超音波溶着機を動かすための自動溶着ステージを制作したので、共有します。素人が作成したものであるので、改善できるところも多いと思います。

ファイルの説明
公開版.step → CADデータです
.xlsx → 購入部品と３Ｄプリンターで製作した部品一覧です
使い方.pptx → 私の環境で動かすときのマニュアル資料です。参考までに
その他 → プログラムファイルです。.pyと.jsonは全部使ってください。

foot_button.pyは溶着機を足でオンオフできるようにするものです。購入品のエクセルにあるフットペダルが必要です。foot_button.pyは単独で実行するものではなくmain_app.pyに組み込まれています
